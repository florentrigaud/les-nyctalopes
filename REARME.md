# Les Nyctalops - Fiches Pathfinder (JDR)

Un outil simple pour gérer tes personnages Pathfinder.

## Installation
1. Ouvre `index.html` dans ton navigateur.
2. Pour un déploiement complet, configure Supabase et Netlify.

## Fonctionnalités
- Connexion sécurisée avec Supabase
- Création/modification de fiches
- Bouton Wiki pour chaque élément
